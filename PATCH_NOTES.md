## 🎲 NOVA MECÂNICA: SISTEMA DE DADOS

### COMO FUNCIONA:
- **D20** para ataques físicos: 1d20 + bônus de classe
- **D12** para magias: 1d12 + bônus de inteligência
- **D10** para habilidades especiais: 1d10 + nível de habilidade
- **D8** para defesa: 1d8 + bônus de armadura

### CRÍTICOS E FALHAS:
- **Crítico Natural (20)**: Dano ×2
- **Falha Crítica (1)**: Perde turno
- **Sucesso Crítico (18-19)**: +25% dano
